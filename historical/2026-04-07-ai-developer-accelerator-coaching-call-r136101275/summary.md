## Topics

### Eric's "Kira" Data Analysis App

  - [**Problem:** Non-technical teams are blocked from accessing data warehouses, creating a bottleneck with data teams.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1581.0)
  - [**Solution:** "Kira" provides a natural language interface to query data warehouses (BigQuery, Redshift, Snowflake, etc.).](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1140.0)
  - [**Key Features:**](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1260.0)
      - [**RAG (Retrieval-Augmented Generation):** Creates a knowledge base from selected tables/columns to improve query accuracy.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1260.0)
      - [**SQL Transparency:** Shows the generated SQL query, which is view-only to prevent data modification.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1633.0)
      - [**Hybrid Analysis:** Joins uploaded CSV/Excel files with warehouse data for combined analysis.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1706.0)
  - [**Differentiator:** Direct Slack integration, allowing users to query data from any channel.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1994.0)
      - [**Security:** Access is managed at the workspace level, restricting users to specific tables.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=1785.0)
  - [**Business Need:** Seeking funding for high Google Cloud costs and competitive market research.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=2493.0)

### AI Development Workflows & Trends

  - [**Ty's User-Driven Development Loop:**](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=5630.0)
      - [**Process:** Users submit feedback (screenshots, videos) → Claude Code clarifies requirements to 85% confidence → Claude Code builds the feature → Codex tests it → Ty reviews and merges the PR.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=5630.0)
      - [**Benefit:** Enables rapid, user-owned development with minimal developer overhead.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=5700.0)
  - [**Patrick's Enterprise SDLC Model:**](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=6121.0)
      - [**Process:** Meeting transcripts are digested into an intent document → Claude Code splits the document into independent workstreams → Developers use Claude Code to plan, implement, and merge their workstreams.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=6159.0)
      - [**Benefit:** Scales agentic development for enterprise teams by enabling parallel, independent work.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=6121.0)
  - [**Claude Code + Codex Plugin:** An official plugin allows Claude Code to offload code review and "rescue" tasks to Codex (GPT-4).](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=5146.0)
      - [**Benefit:** Saves tokens and improves code quality by leveraging Codex's strengths, all within Claude Code's interface.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=5193.0)
  - [**Gemma 4:** Google's new open-source model is highly performant and can run locally (e.g., on a phone or Raspberry Pi).](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=4634.0)
      - [**Use Case:** Patrick plans to use it for a local, secure RAG database of personal health data.](https://fathom.video/share/yiP3fpzB5Nx-NSCVhBePK7sJMBFDfrxy?tab=summary&timestamp=4783.0)
